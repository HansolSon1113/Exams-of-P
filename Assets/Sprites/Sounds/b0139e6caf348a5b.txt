타이틀 -> 밤
타이틀: 타이틀 배경음
if 스타트 버튼을 누르면
fade out(점점 어두워짐) 시작 + 효과음2 + 타이틀 배경음 점점 작아짐
	if 완전 흑백 화면이 되면
	fade in 시작(점점 밝아짐)
	타이틀 배경음 끝, 밤 배경음악 점점 커짐, 효과음 2 유지
if 흑백화면이 끝난다면(완전히 밝아지면)
밤 배경음악 100퍼센트, 효과음 2 끝  

밤
밤: 밤 배경음악
if 카드 레이아웃 슬라이드 버튼을 누르면
	효과음 1
if 카드 레이아웃에 있는 카드를 선택지에 올려놓았을 때
	효과음 6
if 카드 덱 버튼을 눌렀을 때
	효과음 7
if 밤에서 낮으로 버튼을 누르면
톰과 제리 애니메이션 시작 + 효과음2 + 밤 배경음 점점 작아짐
	if 완전 흑백 화면이 되면
	밤 배경음 끝, 낮 배경음악 점점 커짐, 효과음 2 유지
if 흑백화면이 끝난다면(톰과 제리 애니메이션 끝나는 순간)
낮 배경음악 100퍼센트, 효과음 2 끝 

낮
낮: 낮 배경음악
if 카드 드로우 버튼을 누르면
	효과음 1 
if 낮에서 밤으로 버튼을 누르면
톰과 제리 애니메이션 시작 + 효과음2 + 낮 배경음 점점 작아짐
	if 완전 흑백 화면이 되면
	낮 배경음 끝, 밤 배경음악 점점 커짐, 효과음 2 유지
if 흑백화면이 끝난다면(톰과 제리 애니메이션 끝나는 순간)
밤 배경음악 100퍼센트, 효과음 2 끝 

노멀엔딩
if 마지막 낮을 보내고 밤으로 가기 버튼을 누르면
fade out(점점 어두워짐) 시작 + 낮 배경음 점점 작아짐
	if 완전 흑백 화면이 되면
	fade in 시작(점점 밝아짐)
	낮 배경음 끝, 노멀 엔딩 컷씬 배경음악 시작

if 노말 엔딩 컷씬 배경음악이 끝나기 직전
fade out(점점 어두워짐) 시작
	if 완전 흑백 화면이 되면
	fade in 시작(점점 밝아짐)
	노멀 엔딩 배경음악 점점 커짐
if 흑백화면이 끝난다면(완전히 밝아지면)
노말 엔딩 배경음악 100퍼센트

if 타이틀로 누르면
fade out(점점 어두워짐) 시작 + 노멀 엔딩음 점점 작아짐
	if 완전 흑백 화면이 되면
	fade in 시작(점점 밝아짐)
	노멀 엔딩 배경음 끝, 타이틀 배경 배경음악 점점 크게	
if 흑백화면이 끝난다면(완전히 밝아지면)
타이틀 배경음악 100퍼센트

if 다시하기로 누르면
fade out(점점 어두워짐) 시작 + 노멀 엔딩음 점점 작아짐
	if 완전 흑백 화면이 되면
	fade in 시작(점점 밝아짐)
	노멀 엔딩 배경음 끝, 밤배경음 점점 크게	
if 흑백화면이 끝난다면(완전히 밝아지면)
밤 배경음 100퍼센트
